##FinanceControl - это мобильное Android-приложение для комплексного учета расходов и доходов малого бизнеса, анализа финансовой активности и управления бюджетом.

Приложение позволяет пользователю:
- вести учет финансовых операций;
- анализировать доходы и расходы;
- управлять категориями;
- использовать разные валюты;
- настраивать внешний вид приложения.

Проект разработан в рамках учебной дисциплины по мобильной разработке.

## Основной функционал
### 1. Авторизация и роли пользователей
- Регистрация и вход в систему
- Разграничение ролей (Admin / Manager / User)
- Хранение сессии пользователя
- Автоматический вход

### 2. Работа с операциями
- Добавление доходов и расходов
- Выбор категории
- Удаление одной или всех операций
- Сортировка и фильтрация
- Свайп для удаления операций

### 3. Категории
- Добавление категорий (для администратора)
- Просмотр списка категорий
- Разделение на доходы и расходы

### 4. Аналитика
- Просмотр статистики за период
- Графики доходов и расходов
- Средний доход и расход
- Отображение данных по выбранной валюте

### 5. Валюты
- Поддержка валют:
  - Рубли (RUB)
  - Доллары (USD)
  - Евро (EUR)
  - Турецкие лиры (TRY)
- Автоматическая конвертация
- Сохранение выбранной валюты

### 6. Настройки
- Выбор языка (Русский / Английский / Турецкий)
- Выбор темы (Светлая / Темная)
- Отображение роли пользователя
- Выход из аккаунта

## Технологический стек
- **Язык программирования:** Kotlin
- **Платформа:** Android
- **UI:** Material Design 3
- **Архитектура:** MVVM
- **Навигация:** Navigation Component
- **База данных:** Room
- **Асинхронность:** Coroutines
- **Графики:** MPAndroidChart
- **Хранение настроек:** SharedPreferences

git clone https://github.com/Dinar413/Mobilki.git

## Инструкция по использованию

### Первый запуск
1. Откройте приложение
2. Зарегистрируйтесь или войдите в аккаунт
3. После входа откроется главный экран

### Добавление операции
1. Нажмите кнопку "+"
2. Выберите тип операции (доход / расход)
3. Введите сумму и категорию
4. Сохраните операцию

### Просмотр аналитики
1. Перейдите во вкладку "Аналитика"
2. Выберите период
3. Посмотрите график и статистику

### Настройки
1. Откройте вкладку "Настройки"
2. Выберите валюту, язык и тему
3. При необходимости выйдите из аккаунта


## Безопасность данных
- Данные хранятся локально в базе Room
- Пароли не сохраняются в открытом виде
- Используется изоляция пользовательских данных
- Доступ к функциям ограничен ролями
